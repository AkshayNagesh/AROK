## üîç Quick Reference

### Key Concepts
- **Intent-Aware:** Understands what you're doing (coding vs watching media)
- **Memory Governor:** Manages memory allocation proactively
- **Auto-Suspend:** Automatically pauses low-priority processes
- **Zero-Latency AI:** Fast heuristic-based scoring (no API calls)
- **Demo Mode:** Simulated high memory pressure for presentations

### Key Files
- `AROKApp.swift` - App entry point
- `AppState.swift` - State management
- `ContentView.swift` - UI
- `SystemMonitor.swift` - Metrics collection
- `ProcessIntervener.swift` - Process control
- `IntentEngine.swift` - Mode scoring
- `AIAdvisor.swift` - Enhanced scoring
- `DemoMode.swift` - Demo simulation

### Key Commands
- `CMD+B` - Build
- `CMD+R` - Run
- `CMD+Shift+D` - Toggle demo mode
- `CMD+Shift+N` - Toggle demo mode (fallback)
