# AROK Quick Start Guide

## üöÄ Get Started in 5 Minutes

### Step 1: Install Xcode
1. Open Mac App Store
2. Search "Xcode"
3. Click "Get" (free, ~15GB download)
4. Wait for installation

### Step 2: Open Project
```bash
cd /path/to/AROK
open AROK.xcodeproj
```

### Step 3: Build & Run
1. Press `CMD+B` to build
2. Press `CMD+R` to run
3. Look for CPU icon in menu bar (next to battery)

### Step 4: Test It
1. Click menu bar icon
2. Switch between Build/Chill/Focus modes
3. Watch RAM gauge update
4. Try suspending a process
5. Press `CMD+Shift+D` for demo mode
